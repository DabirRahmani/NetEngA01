نیازه که اول از هر دو فایل خروجی بگیرید. هیچ ورودی برای برنامه در نظر نگرفتم.

میتونید دو تا کلاینت اجرا کنید ولی نباید توی یک path باشن. و فایل خروجی توی همون path ساخته میشود
